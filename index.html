Sobre mim

Olá. Meu nome é Estafani e tenho 31 anos. Formei-me em 2019 no curso de Análise e Desenvolvimento de Sistemas pela Universidade Positivo ​e atualmente sou estudante do curso de Design Gráfico na UTFPR.

Criativa e com minha veia artística gritando alto, ao longo do meu período de formação, a parte que mais me atraía durante o desenvolvimento de projetos avaliativos das disciplinas era pensar e criar telas.

Já durante o período de estágio tive oportunidade de participar de cada etapa da migração de uma aplicação web, que utilizava Java Script e Jquery, para o framework Vue.js. Além disso, também atuei na construção de novas soluções back-end e front-end dentro do projeto, fazendo integração de APIs, usando as tecnologias:
.Net MVC, SQL server, Vue.js, Bootstrap, Git.

Com um bocado de sorte, muita dedicação, curiosidade e facilidade de aprendizado evoluí rapidamente dentro do mercado e hoje atuo como desenvolvendora front-end pleno, buscando sempre criar soluções fluídas para os usuários, aplicando os conceitos de UI e UX, levando em conta o contexto de cada projeto.

Sei que a jornada é longa, com muitos obstáculos e ainda há muito conhecimento a ser adquirido. Por isso não tenho pressa e estou sempre pronta para seguir outros rumos e aberta para participar de novos projetos.

Tecnologias

HTML

HTML é uma linguagem de marcação utilizada na construção de páginas web

CSS

É um mecanismo para adicionar estilo a um documento web, definindo como serão exibidos os elementos contidos no código.

JavaScript

É uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

Vue.js

É um framework JavaScript de código-aberto, focado no desenvolvimento de interfaces de usuário.

NPM

É um gerenciador de Pacotes do Node, amplamente usado por desenvolvedores JavaScript para compartilhar ferramentas, instalar vários módulos e gerenciar suas dependências.

Axios

Biblioteca com o propósito de fazer requisições ajax, consumindo APIs Rest.

Bootstrap

Framework para desenvolvimento componentes de interface, baseado em modelos de design para a tipografia, melhorando a experiência do usuário.

Git

Sistema de controle de versões gratuito e de código aberto, com excelente suporte para ramificar, mesclar e reescrever a história de um repositório.

C#

Linguagem de programação, multiparadigma, de tipagem forte, desenvolvida pela Microsoft como parte da plataforma .NET. A sua sintaxe orientada a objetos.

SQL

Linguagem de pesquisa declarativa padrão para banco de dados relacionals
